# 智能家居管理系统 java726

## 项目概述

本项目是基于SpringBoot和Vue的智能家居管理系统，集成了多种技术栈，旨在为用户提供高效、便捷的家居管理解决方案。

## 技术栈

### 后端

- SpringBoot
- SSM（Spring、Spring MVC、Mybatis）
- JSP
- JavaWeb
- MySQL

### 前端

- Html
- Vue
- JavaScript
- Node.js

## 功能模块

1. 用户管理
2. 设备管理
3. 智能控制
4. 数据分析
5. 系统设置

## 系统角色

1. 系统管理员
2. 普通用户

## 运行环境

- 后端：JDK 1.8+、MySQL 5.7+
- 前端：现代浏览器（Chrome、Firefox等）

## 部署步骤

1. 准备后端运行环境（JDK、MySQL）
2. 导入数据库脚本
3. 部署后端项目（SpringBoot应用）
4. 部署前端项目（Vue应用）
5. 启动项目，开始运行

## 目录结构

## 核心亮点

1. 基于SpringBoot和Vue的双端分离架构，易于维护和扩展
2. 集成多种智能家居设备，实现统一管理
3. 界面友好，操作便捷
4. 数据分析功能，帮助用户了解设备使用情况，优化家居环境
